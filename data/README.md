# Data

This project uses the SST-5 fine-grained sentiment classification dataset.

## Task

Each text example is labeled as one of five sentiment classes:

| Label | Sentiment |
|---|---|
| 0 | Very negative |
| 1 | Negative |
| 2 | Neutral |
| 3 | Positive |
| 4 | Very positive |

## Data Availability

The dataset is downloaded within the notebooks using the provided Google Drive links.

Raw data files are not redistributed in this repository.
