# Results

This folder contains the final evaluation results.

## Files

### lstm_results.csv

Classification report for the Bidirectional LSTM model.

### transformer_results.csv

Classification report for the DistilBERT Transformer model.

### model_comparison.csv

Summary comparison of LSTM and Transformer performance.

## Main Finding

The DistilBERT Transformer model achieved stronger performance than the LSTM model in both accuracy and macro F1-score, suggesting that pretrained contextual language models are better suited for fine-grained sentiment classification.
